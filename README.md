## Quick Start

### Prerequisites

- Python 3.12+
- Node.js 22.x

### Run Locally

```bash
# Backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # fill in your keys
uvicorn main:app --reload --port 8000

# Frontend (new terminal)
cd web
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in values:

- `API_BASE_URL`
- `ENVIRONMENT`
- `LOG_LEVEL`

## Deploying to DigitalOcean

This app is designed for [DigitalOcean App Platform](https://www.digitalocean.com/products/app-platform):

```yaml
name: weekender-route-postcards
services:
  - name: api
    github:
      repo: Two-Weeks-Team/weekender-route-postcards
      branch: master
    build_command: pip install -r requirements.txt
    run_command: uvicorn main:app --host 0.0.0.0 --port $PORT
    http_port: 8080
    instance_size_slug: apps-s-1vcpu-0.5gb
    envs:
      - key: GRADIENT_MODEL_ACCESS_KEY
        scope: RUN_TIME
        type: SECRET
      - key: DIGITALOCEAN_INFERENCE_KEY
        scope: RUN_TIME
        type: SECRET
static_sites:
  - name: web
    github:
      repo: Two-Weeks-Team/weekender-route-postcards
      branch: master
    source_dir: web
    build_command: if [ -s package-lock.json ]; then npm ci || npm install; else npm install; fi && npm run build
    output_dir: web/.next
```
